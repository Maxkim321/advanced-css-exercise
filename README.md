三、使用说明 / 3. Usage Guide
克隆本仓库到本地（Clone the repository to your local machine）：
bash
git clone https://github.com/Maxkim321/advanced-css-exercise.git
（替换为你的仓库地址 / Note: Replace the URL with your actual repository link.）
查看练习效果（View exercise results）：进入对应章节文件夹，直接在浏览器中打开 index.html 文件即可。Navigate to the folder of the target chapter, then open the index.html file in a browser.
修改并编译 Sass 文件（Modify and compile Sass files）：
首先全局安装 Sass 环境（First, install the Sass environment globally）：
bash
npm install -g sass
执行监听命令，实现 Sass 到 CSS 的自动编译（Run the watch command to auto-compile Sass to CSS）：
bash
sass scss/style.scss css/style.css --watch
