Advanced CSS & Sass Study Notes & Practice Repository
This repository houses all practice code, project implementations, and knowledge summaries from my learning journey of the Udemy course Advanced CSS and Sass: Flexbox, Grid, Animations and More!. It tracks the complete process of mastering modern CSS techniques, from foundational concepts to advanced applications.

1. About This Repository
   1.1 Learning Source
   Course Name: Advanced CSS and Sass: Flexbox, Grid, Animations and More!
   Course Link: https://www.udemy.com/course/advanced-css-and-sass/
   Core Objective: Master Flexbox, Grid layouts, Sass preprocessor, CSS animations, and responsive design through hands-on exercises to enhance front-end styling skills.
2. Tech Stack
   Core Technologies: CSS3 (Flexbox, Grid, Animations, Transitions, Media Queries)
   Preprocessor: Sass (using SCSS syntax)
   Tools & Environment: VS Code, Live Server (for real-time preview), Git (for version control)
3. Usage Guide
   Clone the repository to your local machine:
   git clone https://github.com/Maxkim321/advanced-css-exercise.git
   (Note: Replace the URL with your actual repository link.)
   View exercise results: Navigate to the target chapter folder and open the index.html file in a browser.
   Modify and compile Sass files:
   First, install the Sass environment globally: npm install -g sass.
   Run the watch command for auto-compilation: sass scss/style.scss css/style.css --watch.
4. Notes
   Continuous Updates: The repository will be updated regularly, with code and notes for each chapter committed after each study session.
   Feedback & Collaboration: If you spot bugs or have optimization suggestions, feel free to communicate via Issues or Pull Requests.
   Note Organization: Knowledge notes compiled during the learning process will be uploaded in Markdown format later for future review and reference.
